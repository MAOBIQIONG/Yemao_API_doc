1.实名认证
    资源定位：
        submitAutRealName
    交互方式：
        post
    请求参数：
        {
            real_name:"真实姓名",
            mobile_phone:"手机号码",
            ID_card:"身份证号码",
            ID_card_handled_front_view:"手持身份证正面照",
            ID_card_front_view:"身份证正面照" ,
            ID_card_back_view:"身份证背面照"
        }
    请求操作：
        提交设计师信息,待审核
    返回参数：
        {
            code:"状态值",
            msg:"返回消息"
        }

2.平台认证
    资源定位：
        submitAutPlatform
    交互方式：
        post
    请求参数：
        {
            real_name:"真实姓名",
            mobile_phone:"手机号码",
            ID_card:"身份证号码",
            university_name:"学校名称",
            certificate_picture:"文凭图片",
            positional_title:"职称证书",
            positional_title_picture:"职称证书"
        }
    请求操作：
        提交设计师学历职称信息,待审核
    返回参数：
        {
            code:"状态值",
            msg:"返回消息"
        }

3.签约设计师
    资源定位：
        submitAutContract
    交互方式：
        post
    请求参数：
        {
            region:"所在地区",
            email:"邮箱地址"
        }
    请求操作：
        提交设计师邮箱,待审核
    返回参数：
        {
            code:"状态值",
            msg:"返回消息"
        }

4.是否认证
    资源定位：
        ifAut
    交互方式：
        get
    请求参数：
        {
            user_id:"用户id"
        }
    返回参数：
        {
            authenticating_state:"认证状态"(见状态说明.txt)
        }
